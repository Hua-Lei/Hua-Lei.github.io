hexo博客使用Gitalk几个注意点

发表于 2020-01-27 阅读次数：

前言

今天又把gitalk集成到博客里了，总的还是很简单的，但还是有几项注意

正文

注册应用

进入链接：https://github.com/settings/applications/new
然后可看见一个表单

Application name，应用名，这个随便
在Homepage URL中填你的网站主页链接
Authorization callback URL，回调地址，填你网站链接就行
注意填网页链接的两栏一定要加上 https:// ！！

相关配置

注册完成后，你会看见两串信息，现在回到主题配置，搜索gitalk： 

gitalk:
enable: true  # 改成true 
github_id:   # 填你的用户名
repo: # 填仓库名，不加https前缀
client_id:  # 两串信息之一，填进去
client_secret:  # 同上
admin_user:  # 填你的用户名

保存然后三步走上传github就好了
Tip：如果你一开始只写有一个博客你是看不见的下面评论区的（这个困扰我很久）你可以点下文章标题，这时网页链接和主页链接不同，如果你可以看见下方评论区就代表你成功了。
